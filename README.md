This project showcases a **Sales Dashboard for a Cafe**, built using **Microsoft Power BI**. It provides interactive visualizations and key insights to help stakeholders make informed business decisions.

# 📊 Dashboard Overview

The dashboard includes:

- ✅ **Total Sales, Profit, and Quantity Sold**
- 📈 **Monthly Sales Trends**
- 🍽️ **Top-Selling Items**
- 🕒 **Peak Sales Hours**
- 👤 **Customer Insights**

# 📁 Files Included

- `cafe sales dashboard.pbix` — Power BI file containing the full dashboard
- `Balaji Fast Food Sales.csv` — Cleaned dataset used for this project
- `README.md` — Project documentation (you are here)

# 🛠 Tools Used

- **Power BI** — Data visualization
- **Excel** — Data cleaning and transformation

# 💡 Insights Gained

- Identified peak sales hours and most profitable product 
- Uncovered trends in seasonal demand and customer behavior
- Enabled data-driven decisions for marketing and inventory
